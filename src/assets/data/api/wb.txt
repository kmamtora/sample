catalog load --filepath {jsonfile}
appconfig package --dir {appconfig}
logo file --filepath Logo.png

## RHEL  catalog bundle
image build --basedir image/rhel --image-repotag {repotag}
image package --image-repotag {repotag} --os rhel7 
catalog save --filepath staging/{jsonfile} --force
sources package --additional {appconfig}
catalog package
